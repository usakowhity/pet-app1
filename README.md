# My AI Pet 🐰🐶🐱

スマホで声や笑顔に反応する、心が通うAIペットアプリ！

「かわいい！」と言うと大喜びで寄ってきて、画面を撫でると甘え、  
「お水！」と言うと水を飲み、「ねんねして」と囁くとおやすみモードに…

あなたの愛ペットと心が通う、究極の癒しをお届けします。

### すぐ遊べる！

このURLをスマホで開くだけ！  
👉 https://usakowhity.github.io/pet-app1/

### 対応ペット

* Usako（白うさぎ・動画豊富）
* Kuro（黒うさぎ・動画豊富）
* Taro（パピヨン子犬）
* Marple（トイプードル）
* Pochi（柴犬子犬）
* Tama（猫）

### 遊び方

* マイクボタンで話しかけ（「ごはん」「お水」「トイレ」「ねんね」など）
* カメラボタンで笑顔を見せると大喜び！
* ペットの画像を指で撫でると甘えます♡
